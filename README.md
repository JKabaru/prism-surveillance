- NetworkX
- Pandas

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/prism-surveillance.git
   cd prism-surveillance
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the dashboard:
   ```bash
   streamlit run src/dashboard/app.py
   ```
